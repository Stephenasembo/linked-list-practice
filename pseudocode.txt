Traverse linked list looking for null;
If node with null on next field is found
  Append the value to the linked list.


Create traversal pointer
Place traversal pointer to the begining of the list.
If node.next === null - base case
  Append
Else traversal pointer = node.next;

traversal function(node, val) {
  if (node.next === null) {
    return value;
  }
  return traversal(node.next)
}
